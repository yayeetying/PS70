Capacitive Sensor to Measure Soil Moisture



Above, you can see me working with a Soil Moisture sensor to detect the amount of moisture in my succulent. According to the output, the moisture levels during that particular time with that level of moisture was around 2500. After doing some research, this is what I found out about how my moisture sensor works: essentially, when you place the 2 prongs (2 electrodes) into the soil, the soil acts as dielectic material, and if the water content in that soil increases, the capacitance between the 2 prongs increases as well. This change in capacitance is converted into some volumetric water content reading (which is the number that we're seeing on our screen).

Here is the code to achieve that: 
              
#define sensorPin A0

void setup() {
  Serial.begin(9600);
}

void loop() {
  Serial.print("Analog output: ");
  Serial.println(readSensor());
  delay(500);
}

int readSensor() {
  int sensorValue = analogRead(sensorPin);  // Read the analog value from sensor
  return sensorValue;             // Return analog moisture value
}

Output Device



Above shows the output device I have. I want to include some LED to the final project as a physical indicator that the plant is thriving and doesn't need water. 

              
#include Adafruit_NeoPixel.h

#define PIN        9 
#define NUMPIXELS 10 
                
Adafruit_NeoPixel strip(NUMPIXELS, PIN, NEO_GRB + NEO_KHZ800);
                
void setup() {
  strip.begin(); // INITIALIZE NeoPixel strip object (REQUIRED)
  strip.show();            // Turn OFF all pixels ASAP
  strip.setBrightness(10); // Set BRIGHTNESS low to reduce draw (max = 255)
}
                
void loop() {
  strip.clear(); // Set all pixel colors to 'off'

  for(int i=0; i<=NUMPIXELS-1; i++) {          
    strip.setPixelColor(i, strip.Color(0, 150, 0));         
    strip.show();
  }
}
